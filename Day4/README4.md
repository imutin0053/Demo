1. Input and Display Array Elements

Take N elements as input and display them.

2. Find Sum and Average of Array Elements

Calculate the sum and average of all elements.

3. Find Maximum and Minimum Element

Find the largest and smallest element in an array.

4. Count Even and Odd Numbers

Count how many even and odd elements are present in the array.

5. Search an Element (Linear Search)

Check whether a given element exists in the array.

6. Reverse an Array

Print the array in reverse order.

7. Copy One Array to Another

Create a copy of an existing array.

8. Find Second Largest Element

Find the second largest element without sorting.

9. Count Positive, Negative, and Zero Elements

Count how many positive, negative, and zero values are present.

10. Check if Array is Sorted

Determine whether the array is sorted in ascending order.

11. Remove Duplicate Elements

Print only unique elements from the array.

12. Frequency of Each Element

Count how many times each element appears.
Example:
[1,2,2,3,1]
Output:

1 → 2 times
2 → 2 times
3 → 1 time
13. Merge Two Arrays

Take two arrays and merge them into a third array.

14. Left Rotate and Right Rotate Array

Rotate the array by K positions.

Example:
[1,2,3,4,5]
Left Rotate by 2 → [3,4,5,1,2]

15. Find Missing Number

Given numbers from 1 to N with one missing number, find the missing number.

Example:
[1,2,3,5]
Output: 4
